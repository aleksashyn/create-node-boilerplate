# create-nodejs-app

**Create NodeJS Application**

### Install
    `yarn install`

### Run

**Development mode**  

    `yarn run start:dev`

**Production mode**  

    `yarn run start`

### Test

**Run test**  

    `yarn test`

**Run test with coverage**  

    `yarn run test:coverage`
